# @reticlehq/protocol

Shared wire contract for [Reticle](https://github.com/reticlehq/reticle): constants, TypeScript types, and zod schemas for every message that crosses the browser ↔ bridge ↔ agent boundary.

You usually don't install this directly — it's a dependency of [`@reticlehq/browser`](https://www.npmjs.com/package/@reticlehq/browser) and [`@reticlehq/server`](https://www.npmjs.com/package/@reticlehq/server). MIT.
