# @iris/protocol

Shared wire contract for [Iris](https://github.com/iris-mcp/iris): constants, TypeScript
types, and zod schemas for every message that crosses the browser ↔ bridge ↔ agent boundary.

You usually don't install this directly — it's a dependency of
[`@iris/browser`](https://www.npmjs.com/package/@iris/browser) and
[`@iris/server`](https://www.npmjs.com/package/@iris/server). MIT.
