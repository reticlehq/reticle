# @syrin/protocol

Shared wire contract for [Iris](https://github.com/syrin-labs/iris): constants, TypeScript
types, and zod schemas for every message that crosses the browser ↔ bridge ↔ agent boundary.

You usually don't install this directly — it's a dependency of
[`@syrin/browser`](https://www.npmjs.com/package/@syrin/browser) and
[`@syrin/server`](https://www.npmjs.com/package/@syrin/server). MIT.
